# 服のコーディネートコマンド

![test](https://github.com/reniker1/robs/actions/workflows/test.yml/badge.svg)

服の色や系統を提案する

## 必要なソフトウェア

* Python
  * テスト済みバージョン: 3.7~3.10

## 実行方法

* 自身のターミナルに
```
 git clone https://github.com/reniker1/robs.git
```
を入力

*  ```robs```のレポジトリで
```
./robs
```
を入力

## 実行結果 

```
季節を入力してください(春、夏、秋、冬)=
```
と表示されたら季節を入力。

```
コーディネート
帽子 ニット帽
トップス 茶色
ボトムス ベージュ
シューズ 茶色
系統 スポーツ
```

## テスト環境

* Ubuntu 22.04
