# 服のコーディネートコマンド

![test](https://github.com/reniker1/robs/actions/workflows/test.yml/badge.svg)

ランダムで服の色や系統を提案する

## 必要なソフトウェア

* Python
  * テスト済みバージョン: 3.7~3.10

## 実行結果

* 自身のターミナルに
```
 git clone https://github.com/reniker1/robs.git
```
を入力

*  ```robs```のレポジトリで
```
　Python3 robs.py
```
を入力

## 実行結果 

```
帽子: ニット
ウェア: 紫
パンツ: ベージュ
シューズ: 赤
系統: カジュアル
```

## テスト環境

* Ubuntu 22.04
